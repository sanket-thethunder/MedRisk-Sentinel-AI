# Project Brief: MedRisk Sentinel AI

## Objective

Create a realistic healthcare data analytics project aligned with a Data Analytics Intern role supporting Risk Adjustment operations, NLP innovation, SQL analysis, and dashboard reporting.

## End-to-End Workflow

1. Generate synthetic member encounters with clinical-style notes, ICD-10 codes, HCC-style categories, payer, region, risk score, and model confidence.
2. Simulate code assignment outcomes: true positives, false positives, and false negatives.
3. Store the data in CSV files and a SQLite database.
4. Train an NLP model to classify clinical notes into risk categories.
5. Calculate SQL-backed precision, recall, F1, audit queue, and monthly trend metrics.
6. Visualize results in a Streamlit dashboard for analyst review.

## Why It Matches The Job Description

- Strong SQL: includes analytical queries for joins-ready healthcare tables, PR calculations, and QA reporting.
- Healthcare analytics: uses ICD-10, HCC-style categories, claims-like encounters, and Risk Adjustment concepts.
- NLP and AI exposure: trains a text classification pipeline on unstructured clinical-style text.
- Dashboarding: delivers BI-style KPI monitoring through Streamlit and Plotly.
- Data quality: includes false positive, false negative, confidence, and audit queue logic.
- Python workflows: uses scripts for data generation, model training, and metrics export.

## Suggested GitHub Description

Healthcare Risk Adjustment NLP analytics project using Python, SQL, scikit-learn, and Streamlit to classify clinical text, reconcile ICD-10 code assignments, calculate precision/recall metrics, and visualize HCC-style operational KPIs.
